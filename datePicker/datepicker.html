Calendario com jQuery!



O JavaScript permite ao usuario selecionar a data apenas no clicar do mouse, para isso o jQuery disponibiliza componentes visuais.
Um deles é o campo calendario datepicker. Para implementa-lo, precisa associa-lo a um input de texto com a função JavaScript.
Exemplo:

<script> $( function() { $( "#datepicker" ).datepicker(); } ); </script> <br> <p>Date: <input type="text" id="datepicker"></p>
